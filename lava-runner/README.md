# lava-runner
*A game for OpenComputers.*

![Screenshot 1](http://i.imgur.com/9aOCQgH.png)

## Developers
* electronic\_steve

## Description
Find the exit, and avoid the lava! That's it.

### Controls
* `W`, `A`, `S`, `D` — move.
* `Tab` — change the graphics mode (the screenshot above demonstrates
  the three of them).
